Interval

  • With the interval, you can control how long your specified BPM will play.
  • Enable the randomizer to randomly choose a value between the minimum and maximum settings.
  • If you don't want a random value, leave the randomizer off and set the Maximum value to your desired amount.
  • Use the 'lock' to lock or unlock the interval.

BPM

  • Control the Tempo using the BPM setting
  • Enable the randomizer to randomly choose a value between the minimum and maximum settings.
  • If you don't want a random value, leave the randomizer off and set the Maximum value to your desired amount.
  • There will be an audible "tick" to the specified beat. More sounds to come at a later date.

Rest

  • Set a rest period between intervals
  • Enable the randomizer to randomly choose a value between the minimum and maximum settings.
  • If you don't want a random value, leave the randomizer off and set the Maximum value to your desired amount.
  • This will be a period between intervals where the interval and BPM do not continue. After the rest period, the interval will reset and continue.
  • To skip using the rest period, set the maximum to "0", or lock the interval.
